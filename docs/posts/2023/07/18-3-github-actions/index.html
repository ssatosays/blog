GitHub Actions: サンプルコード #3

July 18, 2023
Dev
  • .github/workflowsディレクトリに適切な名前のyml/yamlファイルを配置する
  • 以下のサンプルコードではAM06:00(UTC)で定期実行するタスクを定義する
name: daily
on:
  schedule:
    - cron: '00 06 * * *'
jobs:
  daily:
    name: daily
    runs-on: ubuntu-latest
    steps:
      - name: echo
        run: |
          echo '--- date'
          date
          echo '--- date'
